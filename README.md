# AIスタンダード研修 受講アンケート

AI研修参加者向けの美しく、機能豊富なWebアンケートアプリケーション。リアルタイム集計機能と管理画面を備えています。

## 🎯 機能

### 参加者向け機能
- **8つの質問** - 満足度調査、AI印象の変化、活用意欲など
  - スケール質問（1-5段階評価）
  - 複数選択質問
  - 単一選択質問
  - 自由記述テキスト
- **リアルタイムプログレス表示** - アンケート進捗を粘着ヘッダーで表示
- **バリデーション** - 必須項目の自動チェック
- **スムーズなアニメーション** - フェードイン、スケールイン効果

### 管理者向け機能
- **結果集計ダッシュボード**
  - スケール質問の平均値表示
  - 複数選択の横棒グラフ
  - 単一選択の集計結果
  - 自由記述の全件表示
- **CSV出力** - 回答データを Excel などで分析可能
- **データリセット** - 管理者による全データ削除機能
- **リアルタイム更新** - 新しい回答が即座に集計に反映

## 💻 技術スタック

- **React 18.3** - UI フレームワーク
- **Vite 6.4** - 高速ビルドツール
- **CSS-in-JS** - インラインスタイルとグローバルCSS
- **localStorage** - クライアント側データ永続化

## 🚀 セットアップ

### 必要な環境
- Node.js 16.0+
- npm 或いは yarn

### インストール

```bash

npm install

# 開発サーバーの起動
npm run dev

# 本番ビルド
npm run build

# ビルド結果のプレビュー
npm run preview
```

### アクセス

開発中:
- ローカル: `http://localhost:3003/`

## 📖 使い方

### 参加者の利用方法

1. アンケートページにアクセス
2. 8つの質問に順番に回答
   - 必須項目（赤い「必須」マーク）は必ず回答してください
   - 任意項目は記入しなくても送信できます
3. プログレスバーで進捗を確認
4. 「回答を送信する」ボタンで送信
5. ありがとう画面が表示されれば完了

### 管理者の利用方法

1. 右上の「管理画面」ボタンをクリック
2. 回答結果を確認
   - 平均スコアカード - 満足度など4つのスケール質問の平均
   - グラフ - 使ってみたいツール、役立った内容
   - その他の回答 - エキスパート研修への関心、自由記述
3. 「CSV出力」ボタンでデータをダウンロード
4. 「全データをリセット」で初期化可能（確認ダイアログあり）

## 💾 データ保存

- アンケート回答は **localStorage** に保存されます
- ブラウザのキャッシュ削除やシークレットモードでは新規カウントになります
- CSV出力でデータがダウンロード可能です

## 🔒 注意事項

- **データは永続的です**: ブラウザのデータ削除機能を使わない限り、管理画面でリセットするまで保持されます
- **同一デバイス**: 異なるブラウザ/デバイスでアンケートに回答すると、それぞれ別々に集計されます
- **バックアップ推奨**: 重要なデータはCSV出力で定期的にバックアップしてください
